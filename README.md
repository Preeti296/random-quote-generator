# 🌟 Random Quote Generator

A simple and responsive Random Quote Generator built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🎯 Generate random quotes using API
- 🌄 Dynamic background images
- 📋 Copy quote to clipboard
- 📤 Share quote
- 💾 Save favorite quotes (localStorage)
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- API (DummyJSON)

## 🎮 How to Use

1. Click on **New Quote** to generate a quote
2. Click **Copy** to copy the quote
3. Click **Share** to share it
4. Click **Save** to store it
